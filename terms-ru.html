Дата вступления в силу: 3 апреля 2026 г.

2. Описание сервиса

BEU — это приложение для отслеживания процедур ухода за кожей, которое помогает вам организовывать, планировать и контролировать ваши ежедневные ритуалы ухода за кожей. Приложение позволяет создавать процедуры, добавлять средства по уходу за кожей, отслеживать этапы и следить за вашей последовательностью с течением времени.

BEU также предоставляет дополнительные функции на основе ИИ, включая:

  • Сканирование продуктов с помощью ИИ: сканируйте этикетки средств по уходу за кожей с помощью камеры для определения ингредиентов, обнаружения потенциальных конфликтов ингредиентов и получения советов по использованию.
  • Анализ кожи с помощью ИИ: загрузите фотографию вашего лица для получения общей оценки типа вашей кожи и персонализированных рекомендаций по уходу за кожей.
  • Анализ процедур с помощью ИИ: получайте рекомендации, созданные ИИ, для улучшения вашей текущей процедуры ухода за кожей на основе существующих этапов и продуктов.

Все функции ИИ работают на основе сторонних моделей ИИ (см. Раздел 5) и подлежат отказу от ответственности в Разделе 9.

5. Сторонние сервисы

Приложение использует сторонние сервисы, включая:

  • Аутентификация: Apple Sign-In, Google Sign-In и Firebase Authentication для управления учетными записями пользователей.
  • Хранение данных: Firebase Firestore и Firebase Storage (Google LLC) для хранения пользовательского контента.
  • Обработка ИИ: Firebase AI Logic на базе Google Gemini (Google LLC) для обработки фотографий и текста, отправленных через функции ИИ Приложения, включая сканирование продуктов, анализ кожи и анализ процедур. Когда вы используете функции ИИ, ваши фотографии и/или текстовые данные передаются на серверы Google для обработки моделями ИИ.

Эти сервисы регулируются их собственными условиями и политиками конфиденциальности. BEU не несет ответственности за практики, доступность или точность этих сторонних сервисов.
